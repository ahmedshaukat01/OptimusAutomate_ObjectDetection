Object Detection App

Overview

This project is a Streamlit-based web application that performs object detection using the pre-trained YOLOv8 model.

Features

- Upload an image
- Detect multiple objects
- Display bounding boxes
- Show object labels
- Display confidence scores
- Simple Streamlit web interface

Technologies Used

- Python
- Streamlit
- Ultralytics YOLOv8
- OpenCV
- Pillow
- NumPy

Live Demo

https://optimusautomatetask3-o8qzmtxu3htluyksoj73tn.streamlit.app/

How to Run

1. Install the required dependencies.
2. Run:
   streamlit run TASK3.py
3. Upload an image to detect objects.
